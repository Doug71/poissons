Afin d'utiliser notre logiciel de fiches sur les poissons, il faudra d'abord installé un petit logiciel qui émulera un serveur WEB :

Ces logiciel, suivant la plateforme, sont:

Wamp (Windows): Vous pouvez le télécharger sur ce lien : http://www.wampserver.com/.

MAMP (MacOs): Vous pouvez le télécharger sur ce lien : https://www.mamp.info/en/downloads/

XAMPP(Linux, MacOs ou Windows) : https://www.apachefriends.org/fr/index.html

Une fois installé, un dossier WAMP apparaît à la racine, C:\ (Par défaut). Ouvrez ce dossier et lancer l’application : wampmanager.
Une fois ouvert une icône en bas à droite de votre bureau apparait, il est symbolisé par un W (vert quand tout va bien).

Ensuite, enregistrer vos fichiers télécharger dans le dossier : C:\wamp\www\poissons

Voilà, c'est presque finit, maintenant, il suffit d'aller sur votre nagigateur préféré et de taper l'URL http://localhost/poissons


C'est finit, aprécier votre nouveau logiciel.
