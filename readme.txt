Afin d'utiliser notre catalogue sur les poissons, il faudra d'abord installer un petit logiciel qui émulera un serveur WEB :

Ce logiciel, suivant la plateforme, est:

Wamp (Windows): Vous pouvez le télécharger sur ce lien : http://www.wampserver.com/.

MAMP (MacOs): Vous pouvez le télécharger sur ce lien : https://www.mamp.info/en/downloads/

XAMPP(Linux, MacOs ou Windows) : https://www.apachefriends.org/fr/index.html

Une fois installé, un dossier WAMP apparaît à la racine, C:\ (Par défaut). Ouvrez ce dossier et lancez l’application : wampmanager.
Une fois ouvert une icône en bas à droite de votre bureau apparait, elle est symbolisée par un W (vert quand tout va bien).

Ensuite, enregistrez vos fichiers téléchargés dans le dossier : C:\wamp\www\poissons

Voilà, c'est presque fini, maintenant, il suffit d'aller sur votre navigateur préféré et de taper l'URL http://localhost/poissons


C'est fini, apréciez votre nouveau logiciel.
